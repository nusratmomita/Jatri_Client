<h1>Welcome To Jatri🚗</h1>
<h2>Want to go on a trip but worried about renting cars? We have got you! </h2>
<p>Jatri is a car rental web service that allows users to book and list their preferred cars. Not just this, they can update and delete anytime they want.</p>

<h2>📝Core Features</h2>
<ul>
  <li>Browse all the cars in a card interface.</li>
  <li>Visualize cars added by the user in a table format, which allows users to update, view, and delete cars easily.</li>
  <li>Implemented searching and sorting functionality based on car price and the date the car was added.</li>
  <li>My Booking page shows the user's booked cars, and here they can confirm the booking by updating the dates.</li>
  <li>My Cars page shows the user's list of cars, and here they can update car information, delete a car if they want to.</li>
  <li>Implemented the authentication system with Email/Password and Google login and registration system.</li>
</ul>
